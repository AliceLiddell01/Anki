# Anki Deck Repository

Хранилище моих колод для **Anki**.  
Все колоды здесь хранятся в формате **JSON**, чтобы можно было отслеживать изменения через Git — в отличие от оригинального формата `.apkg`, который этого не позволяет.

---

## Установка расширения

1. В Anki откройте: **Ctrl + Shift + A → Загрузить дополнение**  
2. Введите код: **1788670778** (расширение [CrowdAnki](https://ankiweb.net/shared/info/1788670778))  
3. Перезапустите Anki после установки.

---

## Импорт колоды

1. Скачайте нужную колоду из репозитория.  
2. В Anki выберите: **Файл → CrowdAnki: Import from disk**  
3. Укажите папку с колодой.

### Настройки импорта:

- **Personal Fields** — выделите поля, которые *не нужно обновлять* из оригинальной колоды (например, если вы в них добавляли свои заметки).  
- **Tag Cards** — при включении добавит к импортируемым карточкам указанные теги.  
- **Do Not Move Existing Cards** — при включении CrowdAnki не будет перемещать старые карточки, даже если структура колоды изменилась.

**Рекомендация:**  
Перед импортом экспортируйте вашу текущую учебную колоду — это позволит безопасно откатиться в случае ошибок или конфликтов.

---

## Экспорт колоды

1. Нажмите на шестерёнку справа от числа карточек к просмотру в нужной колоде.  
2. Выберите **Экспортировать**.  
3. Укажите формат **CrowdAnki JSON**.

---

## Как работает обновление колод

При импорте в уже существующую колоду:

1. **Неизменённые карточки** — сохраняются как есть (прогресс не теряется).  
2. **Обновлённые карточки** — заменяются новыми версиями, при этом прогресс сохраняется.  
3. **Удалённые карточки** — удаляются и у вас.  
4. **Новые карточки** — добавляются с нулевым прогрессом.
